# zh-pgemu-notes

## 项目简介

zh-pgemu-notes 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面内容不针对任何特定网站或域名，仅作为技术笔记、学习资料或功能演示的静态页面集合。  
仓库本身不承担任何线上服务承诺，仅提供页面文件的版本化存储与浏览入口。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
├── pages/             # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/            # 页面依赖的静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要页面目录，每个 HTML 文件为一个独立页面，相互之间无强关联。
- `assets/`：页面所需的共用或独立资源文件，按类型存放。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开使用。
- 页面内容可能涉及技术笔记、工具演示、学习记录等，不保证持续更新或维护。
- 页面之间无跳转依赖，可单独访问或引用。

## 维护说明

- 本仓库主要用于个人归档与分享，不承诺对页面内容进行实时更新或技术支持。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正，但请注意保持内容的中立与简洁。
- 所有页面文件均以 `.html` 扩展名保存，编码统一为 UTF-8。

## 使用方式

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/your-username/zh-pgemu-notes.git
   ```
2. 在 `pages/` 目录下找到感兴趣的 HTML 文件，直接用浏览器打开即可查看。
3. 如需部署至静态托管服务，可将整个仓库内容上传至对应平台。
